# eDNA Research Portal - 디자인 철학 및 구조

## 디자인 선택: 학술적 우아함 (Academic Elegance)

**Design Movement**: 현대 학술 출판 디자인 + 과학 데이터 시각화 미학
**Core Principles**:
1. **명확한 정보 계층**: 연구 흐름을 시각적으로 명확하게 표현
2. **신뢰성과 전문성**: 학술적 신뢰도를 높이는 디자인 언어
3. **상호작용성**: 정보 탐색을 직관적이고 즐겁게 만들기
4. **접근성**: 모든 사용자가 복잡한 과학 정보를 이해할 수 있도록 설계

**Color Philosophy**:
- **Primary**: 깊은 청색 (Oceanic Blue, #1e40af) - 과학적 신뢰성과 수생 생태계 연결
- **Secondary**: 자연 녹색 (Forest Green, #15803d) - 생물다양성과 생태 보전
- **Accent**: 따뜻한 주황색 (Warm Amber, #ea580c) - 데이터 강조 및 시각적 활력
- **Background**: 깔끔한 흰색 + 미묘한 회색 톤 - 학술 문서의 전문성
- **Text**: 진한 회색 (Charcoal, #1f2937) - 높은 가독성

**Layout Paradigm**:
- 비대칭 그리드 기반 레이아웃
- 좌측 네비게이션 사이드바 + 중앙 콘텐츠 영역
- 타임라인 시각화로 연구 역사 표현
- 카드 기반 섹션으로 정보 모듈화

**Signature Elements**:
1. **연구 타임라인**: 1980년대부터 현재까지의 eDNA 연구 진화를 시각적으로 표현
2. **데이터 플로우 다이어그램**: 샘플링 → 추출 → 분석 → 해석의 과정을 도식화
3. **인터랙티브 카드**: 각 주요 논문/보고서를 클릭하여 상세 정보 확인

**Typography System**:
- **Display Font**: Playfair Display (학술 출판의 고전적 우아함)
- **Body Font**: Inter (현대적이고 읽기 쉬운 산세리프)
- **Hierarchy**: 
  - H1: 48px, 700 weight (주제)
  - H2: 36px, 600 weight (섹션)
  - H3: 24px, 600 weight (소제목)
  - Body: 16px, 400 weight (본문)

**Brand Essence**:
- **One-liner**: "eDNA 연구를 통해 지구의 생물다양성을 이해하고 보전하는 글로벌 커뮤니티"
- **Personality**: 신뢰할 수 있는, 개방적인, 혁신적인

**Brand Voice**:
- 학술적이면서도 접근 가능한 톤
- 예시: "환경에서 DNA를 읽다: 생명의 흔적을 따라가는 과학"

**Wordmark & Logo**:
- 로고: DNA 나선 모양과 물방울을 결합한 심볼 (투명 배경, PNG)
- 색상: 깊은 청색 (#1e40af)

**Signature Brand Color**: 깊은 청색 (#1e40af)

## 웹사이트 구조

### 페이지 구성:
1. **홈 (Hero + Overview)**: 
   - 매력적인 히어로 섹션
   - eDNA의 정의 및 중요성 요약
   - 주요 통계 (연구 논문 수, 글로벌 네트워크 등)

2. **연구 흐름 (Research Timeline)**:
   - 1980년대부터 현재까지의 eDNA 연구 진화
   - 각 시대별 주요 성과 및 기술 발전

3. **최신 동향 (Latest Trends)**:
   - 2024-2026년의 주요 연구 동향
   - 글로벌 모니터링 네트워크
   - 기술적 진보 (AI, 공중 eDNA 등)

4. **방법론 (Methodology)**:
   - 샘플링, 추출, 분석 단계별 설명
   - 표준화 가이드라인
   - 한계점 및 해결 전략

5. **레퍼런스 (References)**:
   - 모든 인용 논문 및 보고서 목록
   - 원문 링크 및 DOI
   - 다운로드 가능한 PDF 링크

### 상호작용 요소:
- 타임라인 클릭 시 상세 정보 모달
- 논문 카드 호버 시 요약 정보 표시
- 섹션 간 부드러운 스크롤 네비게이션
- 모바일 반응형 디자인

## 시각적 자산 계획:
1. **히어로 배경**: 해양 생물다양성 또는 DNA 나선 추상 이미지
2. **타임라인 아이콘**: 각 시대별 특징을 나타내는 커스텀 아이콘
3. **데이터 시각화**: 연구 논문 수 증가 추이 차트
4. **로고**: DNA 나선 + 물방울 심볼
